p tag

p tag는 문단을 나타내는 문서의 기본이며 가장 많이 사용하는 tag

HTML로 작성한 문서는 한 tag 내에서는 Enter(줄바꿈)이 빈칸 한칸으로만 인식된다 Enter를 아무리 눌러도 Source에서는 보이지만 Web 화면에서는 한줄로 나타날 뿐이다

자동 줄바꿈(Auto word Wrap)

한 tag 내에 작성된 Text는 한줄로 특별한 tag를 사용하지 않으면 한줄에 모두 표현된다. 다만, 실제 Web 화면에서 화면 폭(Width)보다 큰 Text 라인은 자동으로 줄바꿈이 된다(Auto wrap)

강제 줄바꿈(brake rine)

문단 내에서 강제로 줄바꿈을 하고싶다
문단내에서 강제 줄바꿈을 하고 싶을때는 br tag 를 self closing으로 붙여준다.
r gtaf로 강제 줄바꿈을 한 경우엔 문단을 나눈것과 줄 간격이 한 문단내의 다른 Text 라인과 같다.
